Tarefa 09: Lista de Postagens

Utilizando a plataforma Expo Snack, implemente uma aplicação que exibe uma lista de postagens.
Cada item da lista deve exibir um Título, o Resumo, a Quantidade de curtidas e a Quantidade de compartilhamentos de cada postagem.
O usuário poderá selecionar uma das postagens e, com isso, uma nova tela exibe detalhes da postagem selecionada (use navegação).
A aplicação deve estar componentizada.
